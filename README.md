# Hooksy.click

AI-сервис для генерации вирусных постов — «Попадутся на крючок одним кликом».

## Запуск (dev)

**Терминал 1 — API:**
```bash
python server.py
```

**Терминал 2 — UI:**
```bash
npm install
npm run dev
```

Откройте http://localhost:5173 (прокси `/api` → Flask :5000)

## Production

```bash
npm run build
python server.py
```

Откройте http://localhost:5000
